6.2 — How It Works

🎮 The Game Loop

1

User signs up and joins or creates a team (Class A, Friends, Community, etc.)

2

User completes real-life eco actions — recycling, planting, saving water — and logs them with optional photo proof

3

AI verifies the action (image recognition) and credits EcoPoints to the user and their team

4

Points unlock buildings in the shared virtual EcoCity — trees, parks, solar plants, city halls

5

Team leaderboard updates in real-time — competition drives more real-world action

🏙️ City Growth System

Buildings unlock progressively as teams accumulate EcoPoints:
🌳 Tree50 pts
🌿 Park100 pts
🏠 Eco Building200 pts
🏛️ City Hall350 pts
⚡ Solar Plant500 pts
🌐 Global Hub700 pts

⚙️ API Access (Bonus)

EcoCity Builders offers a REST API for schools, NGOs, and partner organizations to integrate eco tracking into their own platforms.

# POST — Log an eco action
POST https://api.ecocitybuilders.uz/v1/actions
{
  "user_id": "u_1234",
  "action": "recycle_plastic",
  "team_id": "team_alpha",
  "proof_url": "https://..."
}
# GET — Leaderboard
GET https://api.ecocitybuilders.uz/v1/leaderboard?limit=10
